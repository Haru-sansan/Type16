#  動作環境
・OS：Windows 10 / 11（64bit）
・CPU：Intel Core i5 相当以上
・メモリ：8GB以上
・ストレージ：空き容量 2GB以上
・解像度：1920×1080 推奨
・その他：スピーカー / イヤホン必須
